MCU-up-crawler
=========

## How to setup this project?
1. using pip
    - Requirements
        - Python=3.8
    - `pip install -r requirements.txt`
2. using Conda(Recommend)
    - Requirements
        - conda
    - create ENV 

        `conda env create --file mcu-up-env.yml`
    - Remove ENV
        ```
        conda deactivate
        conda env remove --name MCU-up
        ```
